# SecurePay-FinTech-Platform
SecurePay FinTech Platform - A production-grade payment gateway and banking core system built using Spring Boot microservices, Kafka event-driven architecture, CQRS, Event Sourcing, and distributed tracing.

# SecurePay Payment Gateway

A distributed payment gateway inspired by Stripe, Razorpay, and PayPal.

## Features

- Merchant Onboarding
- Customer Authentication
- Order Management
- Payment Processing
- Banking Accounts
- Double Entry Ledger
- Transaction History
- Fraud Detection
- Merchant Settlement
- Notifications
- Audit Logging
- Distributed Tracing
