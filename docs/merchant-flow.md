# Merchant Onboarding

## Scenario

TechStore wants to accept online payments.

Before accepting payments, TechStore must register with SecurePay.

Merchant submits:

- Business Name
- Trade License
- Bank Account
- Contact Details

After verification SecurePay generates:

Merchant ID : MER12345

API Key : sk_test_xxxxxx

Now TechStore can integrate SecurePay APIs.
